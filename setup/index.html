Só sua, não é salva em lugar algum. Usada para decriptar a host key.

Por que só a host key é cifrada?

A host key tem permissão de publicar, quem tiver acesso pode assumir o controle. Por isso ela passa por criptografia (PBKDF2 + AES-GCM, direto no navegador) antes de ir pro repositório, e só é decriptada na hora que você digita a senha pra apresentar.

A viewer key só tem permissão de assinar (escutar). Mesmo estando exposta no código-fonte da página, não permite fazer nada além de acompanhar uma apresentação.
